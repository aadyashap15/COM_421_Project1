Introduction

This template provides an example of each structural and styling element used in this project. Follow the structure shown here to add new modules, using existing classes and tags to maintain consistency.

Sidebar Title

Sidebars can be used for additional notes or important information. They are styled to stand out from the main content.

Instructions for Adding Content

  • Use <h2> for main section headings.
  • Use <h3> for subsection headings and <h4> for subheadings in sidebars.
  • Place images within <figure> tags with captions inside <figcaption> for proper styling.
  • Ensure all content links to the top-level navigation and follows HTML5 validation rules.

Steps for Adding a New Module

  1. Duplicate this template to create a new module page.
  2. Rename the file appropriately, and add the link to your navigation bar.
  3. Modify the content, headings, images, and other elements as needed for the new module.
  4. Ensure the styles are consistent with the template and existing pages.
