How to Organize and Analyze Research Data

This lesson is for students organizing research data. It covers Excel formatting and PASW analysis.

Preparation

You’ll need Microsoft Excel and PASW. Familiarity with Excel, statistics, and research design is recommended.

Format Data for Analysis

Follow along with ToolBox_DataLesson.xls to organize variables and prepare data for PASW.

Understanding Dichotomous Variables

Using "0" and "1" to represent categorical values helps in isolating characteristics during analysis.

Run Descriptive Statistics

Use Excel’s Analysis ToolPak to perform descriptive statistics and analyze data characteristics.

Interpreting Descriptive Statistics

High variance in certain values can suggest insights, like segments of users with differing views on usability.

Run Inferential Statistics

Upload data into PASW for inferential tests. Use Analyze menu options to explore tests suitable for your variables.

Choosing the Correct Statistic

Consult resources like UCLA's Choosing the Correct Statistic for guidance on statistical analysis.
